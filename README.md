### Como automatizar testes de segurança usando cypress + github actions
Hoje, no mundo do desenvolvimento de software, é inegavél o valor que um bom teste automizado tem na construção de sistemas com uma alta régua de qualidade. Seja na refatoração ou na construção de novas funcionalidades, um bom teste faz com que tenhamos mais segurança em futuras correções, deixando assim o sistema manutenível. Mas e você já parou para pensar em segurança? Muitos acham que só de testes complexos se vive a segurança e este guia visa ajudar devs e QA's a se previnirem de simples ataques que podem causar grandes prejuízos.
